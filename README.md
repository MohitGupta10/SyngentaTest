

# Library-Management-System

A simple online library management system built with MongodDB, Express.js and Node.js. 
## Techonologies used in this application

### Front-end

1. HTML5
2. CSS3
3. BOOTSTRAP 4
4. jQuery

### Back-end

1. MongoDB
2. Express.js
3. Node.js
4. Passport.js


## Functionalitites

Whole app is divided into three modules.

- Admin
- User
- Browse books
